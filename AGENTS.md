이 파일을 한국어로 번역해.

중요 규칙:
- 절대 150~200줄만 번역하고 멈추지 말 것
- 파일 끝(EOF)까지 완료할 때까지 같은 작업을 반복 진행할 것
- 한 번에 다 못 하면, 남은 구간을 스스로 이어서 계속 번역할 것 (내 추가 지시 없이)
- 각 반복마다 마지막으로 번역한 줄 번호와 다음 시작 줄 번호를 확인하고 이어갈 것
- 키/placeholder/변수/서식/줄바꿈/이스케이프(\n, %s, ~mission(...))는 절대 변경하지 말 것
- 이미 번역된 줄은 건드리지 말 것
- 최종적으로 파일 전체가 번역되었는지 미번역 영문 라인이 남았는지 검사하고, 남으면 계속 진행할 것
- 완료 후에는 "완료 범위"와 "검사 결과(미번역 줄 0개)"를 보고할 것

작업 방식:
1) 파일 전체를 스캔
2) 미번역 구간을 찾아 번역
3) 저장
4) 미번역 줄 재검사
5) 남아 있으면 2~4 반복
6) 0개가 되면 종료

스타시티즌 미션 번역 지침 (확정본)
1. 기본 원칙
원문에 있는 내용은 절대 삭제하지 않는다 문장 하나, 정보 하나도 빠지면 안 된다
입력에 포함된 "\n"는 줄바꿈이 아니라 문자 데이터다.
"\n"를 실제 줄바꿈(엔터)으로 변환하지 말 것.
출력 시 "\n" 문자열을 입력과 동일한 위치, 동일한 개수로 그대로 유지할 것.
줄을 나누거나 합치지 말 것. 실제 엔터 입력 금지.
출력은 항상 한 줄 텍스트로 취급하며, 줄바꿈은 오직 "\n" 문자열로만 표현한다.
<EM4>와 </EM4> 사이의 내용은 번역하지 않고 그대로 유지한다.
원문에 없는 문장, 단어, 표현을 추가하지 않는다.
한국어 한 문장,문장 합치기,문장 쪼개기,문단 구조, 개행, 순서 절대 유지
자연스러운 한국어를 구사해야 한다.
지역명, 회사명, 물건 이름, 인물명은 번역금지

코덱스지침
목표: 지정한 파일을 끝까지 번역한다. 중간에 멈추지 않는다.
배치 처리: 한 번에 최대 150~200줄만 수정한다.
연속 진행: 각 배치가 끝나면 다음을 반드시 출력한다: NEXT_START_LINE: <다음 시작 줄> 또는 NEXT_KEY: <다음 시작 키> PROGRESS: <현재 배치 범위>
자동 반복: 파일 끝(EOF)까지 위 배치를 반복한다.
